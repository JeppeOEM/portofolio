Three.js galaxy

Visit the site

A little 3d animation i made in three.js after following some tutorials, it is made out of points that has been placed randomly with a for loop and a Math.random() function.
By changing certain values in the for loop i could then do some kind of "galaxy pattern". It moves by it self and and when you move the mouse. The code is on github and it can be run locally with the build tool vite, if you have node.js installed.
